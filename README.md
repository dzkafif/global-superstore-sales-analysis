# Global Superstore Sales Performance Analysis 📊

Proyek analisis data mandiri untuk mengevaluasi performa penjualan dan profitabilitas perusahaan ritel global pada rentang waktu 2011–2014. 

![Dashboard Power BI](dashboard.png)

## 📌 Tentang Proyek
Proyek ini bertujuan untuk mengubah raw dataset berskala besar menjadi wawasan bisnis (*business insights*) yang dapat ditindaklanjuti. Proses mencakup pembersihan data, pemodelan data (relasi tabel), hingga pembuatan *dashboard* untuk memantau metrik utama perusahaan.

## 🛠️ Teknologi & Tools
* **Microsoft Excel:** Data Preparation, Pivot Table, dan KPI Calculation awal.
* **Power BI:** Data Modeling (Star Schema), DAX (Data Analysis Expressions) untuk *Time Intelligence*, dan Data Visualization.

## 📈 Key Performance Indicators (KPI)
Model data yang dibangun berhasil mengekstrak beberapa metrik utama bisnis secara dinamis:
* **Total Sales:** $12.64M
* **Total Profit:** $1.47M
* **Profit Margin:** 11.61%
* **YoY Growth:** 51.54% (Pertumbuhan agregat bisnis yang sangat positif)

## 🔍 Key Insights
Beberapa temuan utama dari analisis data visual ini:
- Tren penjualan perusahaan menunjukkan **peningkatan yang konsisten** setiap tahunnya dari 2011 hingga 2014.
- Region **Central** merupakan penyumbang pendapatan terbesar (mendominasi lebih dari 50% total *sales* di antara region lainnya).
- Produk kategori *smartphone* sangat mendominasi pasar, dengan **Apple Smart Phone** dan **Cisco Smart Phone** sebagai penyumbang *sales* tertinggi.

## 📂 Struktur File
* `Global Superstore.xlsx`: Dataset yang digunakan sebagai sumber kebenaran (*single source of truth*) beserta kalkulasi dasar.
* `Global Superstore.pbix`: File Power BI yang berisi model relasional, *measure* DAX kustom, dan *dashboard* interaktif.
* `dashboard.png`: Tangkapan layar hasil akhir *dashboard* utama.
